# arko-project-1
CORDIC algorithm and bitmap implementation in MIPS assembly. Sine function values are calculated using fixed-point number representation (for greater precision) and drawn on a bitmap. Amplitude and frequency modulation can be applied to the wave, as well as coordinate shifts.
